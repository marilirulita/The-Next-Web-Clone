# The-Next-Web-Clone
This project aims to create a Heat Map of the page SmashingMagazine.com, The page works best on a 1366x768 resolution and using the Chrome browser.



![screenshot](./assets/ScreenShot.png)

## Built With

- HTML5
- CSS
- Font Awesome
- Google Fonts
